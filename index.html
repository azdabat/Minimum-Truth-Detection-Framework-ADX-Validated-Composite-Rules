⬡ The Cousin Technique Doctrine
MITRE ATT&CK provides a taxonomy of adversary techniques — describing what attackers do during an intrusion. ATT&CK models techniques with vertical depth: technique → sub-technique. What it does not model is substrate adjacency — the reality that many ATT&CK techniques represent the same adversary intent executed across different operating system substrates.
For example, lateral movement techniques such as SMB (T1021.002), DCOM (T1021.003), and WinRM (T1021.006) are operationally interchangeable for attackers. An adversary may pivot dynamically between them depending on firewall restrictions, privileges, or endpoint controls in place. Treating these techniques as independent detection problems creates a false sense of coverage — a SOC that has an SMB rule but no DCOM or WinRM rule has a gap an attacker will find.
The Minimum Truth Detection Framework introduces the Cousin Technique Doctrine — a layer that sits on top of ATT&CK and models adjacent techniques as part of a shared attack ecosystem. This enables detection engineers to reason about coverage gaps between related substrates, designing detection strategies that target adversary intent rather than isolated technique identifiers.
MITRE ATT&CK — VERTICAL DEPTH MODEL
  • Technique → Sub-technique hierarchy
  • Each technique treated as independent
  • Coverage mapped by TTP ID
  • No substrate adjacency modelling
  • Rule-per-technique approach
  • Gap: same intent, different substrate = missed
MTDF COUSIN DOCTRINE — SUBSTRATE ADJACENCY LAYER
  • Cousin ecosystem clusters by shared intent
  • Adjacent substrates explicitly modelled
  • Coverage mapped by adversary intent
  • Minimum truth anchors cross-substrate
  • Composite rules catch intent regardless of substrate
  • Coverage gaps between cousins are visible and tracked

MTDF CORE PRINCIPLES
MINIMUM TRUTH ANCHORING
Every composite rule anchors on the irreducible minimum event that must exist for an attack to be structurally real — substrate-first where no command line is visible, intent-first where the primitive implies capability.
COUSIN TECHNIQUE COVERAGE
Adjacent techniques sharing the same adversary intent are modelled as cousin ecosystems. Detection covers the intent cluster, not just the specific TTP — eliminating substrate pivot gaps.
CUMULATIVE CONVERGENCE SCORING
Binary alerting replaced by cumulative scoring. The anchor fires at low confidence; reinforcement signals elevate to high. Noise reduced ~30% in production without suppressing high-risk truth.
30-DAY ATOMIC PRIMITIVE INDEX
Entity-keyed primitive events tracked across 30 days. Defeats temporal deception — C2 jitter, delayed BYOVD staging, slow-burn APT persistence that time-windowed rules cannot address.
ADX-DOCKER VALIDATION PIPELINE
Automated telemetry ingestion pipeline validating composite rules against Empire and Atomic Red Team simulated attack telemetry. All rules are production-validated, not theoretical.
MISP CTI INTEGRATION
Private MISP lab integrating STIX/TAXII feeds into composite rule generation — closing the pipeline from raw IOC ingestion through to validated KQL output aligned to ATT&CK.
